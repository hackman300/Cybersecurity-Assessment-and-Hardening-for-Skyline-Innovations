# Cybersecurity-Assessment-and-Hardening-for-Skyline-Innovations

For this capstone project, I conducted a comprehensive cybersecurity risk assessment and hardening exercise for Skyline Innovations. The scenario involved three real-world threats: a sophisticated spyware Android APK disguised as "Google Service 12.0," targeted phishing campaigns impersonating IT support that compromised employee credentials, and critical misconfigurations in AWS cloud infrastructure.
